이번 글에서는 변수에대해 알아보자.


변수란, 값을 저장하는 공간이다.
변수는 프로그램에서 데이터를 저장하고 조작하는 데 사용된다.
변수는 특정 데이터 유형을 가지며, 해당 유형에 맞는 값을 저장할 수 있다.
예를 들어, 정수형 변수는 정수 값을 저장할 수 있고, 문자열형 변수는 문자열 값을 저장할 수 있다.


변수는 선언과 초기화 과정을 거쳐 사용된다.
변수 선언은 변수의 이름과 데이터 유형을 지정하는 과정이다.
예를 들어, `int age;`는 정수형 변수 `age`를 선언하는 것이다.

변수 초기화는 변수에 처음으로 값을 할당하는 과정이다.
예를 들어, `age = 25;`는 변수 `age`에 정수 값 25를 할당하는 것이다.


변수 선언과 초기화는 한 줄로 작성할 수도 있다.
예를 들어, `int age = 25;`는 변수 `age`를 선언하고 동시에 초기화하는 것이다.


```cpp
#include <iostream>

int main() {
    int age; // 변수 선언
    age = 25; // 변수 초기화
    std::cout << "나이는 " << age << "살입니다." << std::endl;
    return 0;
    // 위 코드에서는 `age`라는 정수형 변수를 선언하고, 25로 초기화한 후, 해당 값을 출력하는 예시를 보여준다.
}
```


변수 초기화는 선언과 동시에 이루어지지 않아도 된다.
그러나 변수 초기화 없이 변수를 사용하려고 하면, 프로그램이 예기치 않은 동작을 할 수 있다.
따라서 변수를 선언할 때는 가능한 한 초기화하는 것이 좋으며, 초기화하지 않은 변수를 사용하기 전에 반드시 값을 할당해야 한다.


변수는 프로그램에서 데이터를 저장하고 조작하는 데 필수적인 요소이다.
