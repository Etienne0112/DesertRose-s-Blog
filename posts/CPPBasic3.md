이번 글에서는 함수에대해 알아보자.


함수는 특정 작업을 수행하는 코드 블록입니다. 함수는 프로그램의 구조를 개선하고,
코드의 재사용성을 높이며, 프로그램을 더 쉽게 이해할 수 있도록 도와줍니다.


함수는 매개변수, 반환값, 그리고 함수 본문으로 구성됩니다.
매개변수는 함수에 전달되는 입력값을 나타내며,
반환값은 함수가 작업을 수행한 후 결과를 반환하는 값입니다.
함수 본문은 함수가 수행하는 작업을 정의하는 코드 블록입니다.


```cpp
#include <iostream>

int add(int a, int b) { // 함수 선언과 정의
    return a + b; // 함수 본문
}

int main() {
    int result = add(5, 3); // 함수 호출
    std::cout << "5 + 3 = " << result << std::endl; // 결과 출력
    return 0;
}
```

위 코드에서는 `add`라는 함수를 선언하고 정의하였다.
`add` 함수는 두 개의 정수 매개변수를 받아서, 그 합을 반환하는 함수이다.
`main` 함수에서는 `add` 함수를 호출하여 5와 3을 더한 결과를 `result` 변수에 저장하고,
그 결과를 출력하는 예시를 보여준다.


해당 코드를 더욱 자세히 알아보자.
int add()라는 함수가 선언됐고,
매개 변수로 int a와 int b가 선언됐다.
함수 본문에서는 a와 b를 더한 값을 반환하는 return문이 있다.


즉 이 함수는 두 개의 정수를 입력받아, 그 합을 반환하는 함수이다.


int main() 함수에서는 add 함수를 5와 3을 입력으로 호출한다.
add 함수는 5와 3을 더한 결과인 8을 반환하여 result 변수에 저장된다.
int result = add (5, 3);은 다르게 말해
int result = return a + b 라고 생각할 수 있다.


그 후에 std::cout을 사용하여 "5 + 3 = 8"이라는 결과를 출력한다.
함수는 람다함수, 재귀함수, 오버로딩 등 다양한 형태로 존재하지만,
이번 글에서는 가장 기본적인 형태의 함수에 대해서만 알아보았다.
